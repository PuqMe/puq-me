# PuQ.me Web App

Next.js mobile-first frontend for `PuQ.me`.

## Screens

- Onboarding
- Login
- Profile creation
- Swipe screen
- Matches
- Chat
- Settings

## Features

- App Router with TypeScript
- Tailwind mobile-first UI
- PWA manifest and service worker shell caching
- Swipe card interaction
- Offline banner and cached routes
- Push-ready client foundation

## Run

```bash
npm install
npm run dev
```
